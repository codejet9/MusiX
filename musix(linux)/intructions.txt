1) sudo apt-get update
2) sudo apt-get install vlc
3) sudo apt-get install gcc
4) Run VLC and make sure it works
5) Open terminal in this folder
6) "pip install --editable ."
7) "sudo pip install --editable ." If you want "musix" as a command executable from anywhere

8) Now go to "/home/USER/.local/lib/python3.9/site-packages/pafy/locate backend_youtube_dl.py" comment lines 53 and 54 which are related like and dislike counts.

9) Get a youtube v3 api key using your google account. You can refer to this video https://youtu.be/N18czV5tj5o

10) Paste the api key in .env file present in this folder. api_key="PASTE_YOUR_API_KEY". If .env file is not visible check show hidden file option

11) Restart the terminal and type "musix" and click enter.
12) You can open terminal in any folder to start the program.

13) For python3.10 users, go to "/home/USERNAME/.local/lib/python3.10/site-packages/prompt_toolkit/styles/from_dict.py" change "from collections import Mapping" to "form collections.abc import Mapping"

Enjoy :)
